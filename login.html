What happened to your results?

By now you have probably realized that there was something funky going on with the quiz! Did you realize that some questions did not have any valid answers? Did you realize that some questions had all valid answers? Based on platforms like TikTok and other social medias, we can have the tendency to take information as is. For example, did you beleive that one of the presidents mentioned had red hair? Because we have faciliated a society where fact-checking is not always at the forefront, I wanted to create a Tactical Media Project that highlights the media's manipulation of this. Under the main umbrella of this project, is a statement on animal rights. Each question (in some way) relates to the US vs Stevens case, one that was widley regarded as the leading case in animal rights. Robert Stevens was convicted for knowingly selling depictions of animal cruelty with the intention of distrubuting, selling and earning commerical gain. His actions were eventually deemed unconstituional in an 8-1 vote. For more resources on this case, please visit the (reliable) sources below:
